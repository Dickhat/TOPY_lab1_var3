# 📖 Руководство пользователя: Методы одномерной оптимизации

Данное программное обеспечение предназначено для нахождения экстремумов функций одной переменной с использованием классических численных методов. Программа предоставляет графический интерфейс для визуализации процесса поиска, анализа итераций и формирования отчетов.

## 🚀 Начало работы

### Системные требования
*   Установленный компилятор **Rust** (через [rustup](https://rustup.rs/)).
*   Библиотеки для графического вывода (в Linux могут потребоваться пакеты `libwayland-dev`, `libx11-dev`, `libasound2-dev`).

### Запуск программы
В корневом каталоге проекта выполните команду:
```bash
cargo run --release
```

## 🛠 Обзор интерфейса

Интерфейс разделен на три логические зоны:

### 1. Панель настроек (Слева)
*   **🎯 Функция**: Выбор целевой функции для оптимизации.
    *   **F1**: $3x - x^3$ (поиск максимума).
    *   **F2**: $(9 - x^2)/(x^2 + 2x + 3)$ (поиск минимума).
*   **🛠 Метод**: Выбор алгоритма (Дихотомия, Золотое сечение или Фибоначчи).
*   **📝 Параметры**: 
    *   `a`, `b` — границы начального интервала неопределенности.
    *   `ε` (эпсилон) — константа различимости.
    *   `l` — допустимая конечная длина интервала.
*   **🚀 Кнопка "Рассчитать"**: Мгновенное вычисление и вывод итогового результата.
*   **🎬 Анимация**: Блок для пошагового просмотра работы алгоритма.
*   **🏁 Итог**: Карточка с финальными координатами $x^*$, значением функции $f(x^*)$ и количеством итераций.

### 2. График функции (Центр)
*   **Визуализация**:
    *   **Голубая линия**: График функции.
    *   **Синие вертикали**: Начальные границы $[a, b]$.
    *   **Красные вертикали**: Конечный интервал неопределенности.
    *   **Желтая точка**: Найденный экстремум.
*   **Управление**: Используйте **колесико мыши** для масштабирования и **зажатую правую кнопку мыши** для перемещения по графику.

### 3. Таблица итераций (Снизу)
Отображает подробные данные каждого шага: номер шага $K$, границы $a_k, b_k$, промежуточные точки $\lambda, \mu$ и значения функции в них.
*   **Интерактивность**: Кликните по номеру шага **K** в таблице, чтобы на графике отобразились границы и расчетные точки именно этой итерации.

## 🎞 Работа с анимацией

Программа позволяет наглядно увидеть, как сужается интервал поиска:
1.  Установите параметры и выберите метод.
2.  Нажмите **▶ Старт**. Линии на графике начнут плавно смещаться к экстремуму.
3.  Используйте чекбокс **"Плавное перемещение"**, чтобы переключаться между скольжением линий и мгновенными прыжками.
4.  Ползунок **"сек/шаг"** регулирует скорость демонстрации.
5.  Ползунок **"Тек. шаг"** позволяет вручную "перематывать" процесс оптимизации.

## 📄 Экспорт отчетов

Программа поддерживает два формата выгрузки данных:

### 1. Текстовый файл (.TXT)
Простой отчет, содержащий итоговый результат и таблицу значений.

### 2. Документ Word (.DOCX)
При нажатии на кнопку **📝 .DOCX** откроется окно настроек:
*   **🎯 Только финальный результат**: Генерирует краткий отчет с одним графиком (итог) и полной таблицей.
*   **🎞 Пошаговый отчет**: Позволяет создать подробную методичку.
    *   Можно указать диапазон шагов (например, с 1 по 10).
    *   **Интервал шагов**: Позволяет выводить, например, каждый 2-й или 5-й шаг, чтобы не перегружать документ.
    *   В отчет вставляются качественные изображения графиков, созданные библиотекой `plotters`.

## ⚠️ Важные замечания
*   **Условие Дихотомии**: При использовании метода дихотомического поиска должно соблюдаться условие $2\epsilon < l$. Если оно нарушено, программа выведет ошибку.
*   **Метод Фибоначчи**: Количество итераций рассчитывается автоматически исходя из заданной точности $l$. Если интервал $[a, b]$ слишком мал, метод может завершиться за 1-2 шага.
*   **Точность**: Все расчеты проводятся с использованием чисел с плавающей запятой двойной точности (`f64`).
