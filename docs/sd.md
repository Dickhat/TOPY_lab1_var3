# solution design

> Это по большей части для осознания работы всего этого произвдеения искусства 🤨

---

Разделим код на 4 части:
- **app.rs** - вся графика на egui тут.

- **logic.rs** Чистые математические алгоритмы. Они ничего не знают о кнопках и графиках. Они принимают числа и возвращают структуру с результатами.

- **models.rs** Общие структуры данных (например, запись об итерации), которые используют и логика, и GUI.

- **main.rs** точка входа в приложение.
