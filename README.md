# Лабораторная работа №1: Методы одномерной оптимизации
### Вариант №3

Данное приложение на языке Rust представляет собой интерактивный инструмент для поиска экстремумов функций одной переменной с визуализацией процесса поиска в реальном времени.

## 🎯 Математическая постановка задачи

В соответствии с вариантом реализовано нахождение экстремумов для следующих функций:

1.  **Нахождение максимума $F_1(x)$:**
    $$F_1(x) = 3x – x^3 , \quad x \in [-3; 3]$$
2.  **Нахождение минимума $F_2(x)$:**
    $$F_2(x) = \frac{9 - x^2}{x^2 + 2x + 3}, \quad x \in [-10; -3]$$

**Реализованные методы:**
*   Дихотомический поиск (Метод дихотомии).
*   Метод золотого сечения.
*   Метод Фибоначчи.

---

## 🚀 Основные возможности программы

*   **Интерактивное управление:** Ввод параметров интервала $[a, b]$, константы различимости $\epsilon$ и точности $l$ с мгновенной валидацией.
*   **Визуализация итераций:**
    *   Построение динамического графика функции.
    *   Отображение текущего интервала неопределенности и расчетных точек ($\lambda, \mu$).
    *   **Анимация:** Плавное или дискретное (пошаговое) воспроизведение процесса сужения интервала.
*   **Интерактивная таблица:** Вывод всех данных итераций в таблицу. При клике на строку таблицы график автоматически переключается на визуализацию этого конкретного шага.
*   **Профессиональный экспорт:**
    *   Генерация текстовых отчетов (`.txt`).
    *   Формирование полноценных документов Microsoft Word (`.docx`) со встроенными графиками и таблицами (поддерживается как итоговый, так и пошаговый отчет).

## 🛠 Технологический стек

*   **Язык:** Rust (Edition 2024).
*   **GUI Фреймворк:** [eframe/egui](https://github.com/emilk/egui) — для создания быстрого и легкого интерфейса.
*   **Графики (Real-time):** `egui_plot`.
*   **Графики (Export):** `plotters` — для рендеринга PNG-изображений высокой четкости.
*   **Работа с документами:** `docx-rs`.
