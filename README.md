# Лабораторные работы по методам оптимизации
Комплексное приложение для исследования численных методов оптимизации

Данное приложение на языке Rust представляет собой интерактивный инструмент для изучения численных методов оптимизации с визуализацией процесса поиска в реальном времени и возможностью переключения между различными лабораторными работами.

---

## 📚 Лабораторные работы

### 🔷 Лабораторная работа №1: Методы одномерной оптимизации
**Вариант №3**

#### 🎯 Математическая постановка задачи

1.  **Нахождение максимума $F_1(x)$:**
    $$F_1(x) = 3x – x^3 , \quad x \in [-3; 3]$$
2.  **Нахождение минимума $F_2(x)$:**
    $$F_2(x) = \frac{9 - x^2}{x^2 + 2x + 3}, \quad x \in [-10; -3]$$

#### 📋 Реализованные методы:
*   Дихотомический поиск (Метод дихотомии).
*   Метод золотого сечения.
*   Метод Фибоначчи.

#### 💡 Возможности:
*   Интерактивное управление параметрами ($[a, b]$, $\epsilon$, $l$)
*   Анимация процесса поиска
*   Интерактивная таблица результатов
*   Экспорт в `.txt` и `.docx` с графиками

---

### 🔶 Лабораторная работа №2: Метод Хука-Джевса с дискретным шагом
**Вариант №3 — Многомерная оптимизация**

#### 🎯 Математическая постановка задачи

1.  **Минимизация $F_1(x_1, x_2)$:**
    $$F_1(x) = -6x_1 - 4x_2 + x_1^2 + x_2^2 + 18$$
    Начальная точка: $X_0 = (1; 1)$

2.  **Минимизация $F_2(x_1, x_2, x_3)$:**
    $$F_2(x) = 4x_1^2 + 3x_2^2 + x_3^2 + 4x_1x_2 - 2x_2x_3 - 16x_1 - 4x_3$$
    Начальная точка: $X_0 = (1; 1; 1)$

#### 📋 Исследуемые аспекты:
*   Поиск экстремума при различных начальных точках
*   Анализ влияния точности $\epsilon = 0.1, 0.01, 0.001$
*   Пошаговое отслеживание итераций (координаты, значения функции, направления)
*   Визуализация траектории поиска на графике с линиями уровня
*   Экспорт результатов с сравнением различных стратегий

---

## 🚀 Основные возможности программы

*   **Переключение между лабораторными работами:**  Две вкладки в верхнем меню для выбора функционала
*   **Интерактивное управление:** Изменение параметров с мгновенной валидацией
*   **Визуализация итераций:**
    *   Динамический график функции
    *   Отображение текущего состояния поиска
    *   **Анимация:** Плавное или дискретное воспроизведение
*   **Интерактивная таблица:** Вывод всех данных итераций, клик переходит на конкретный шаг
*   **Профессиональный экспорт:**
    *   Генерация текстовых отчетов (`.txt`)
    *   Формирование документов Microsoft Word (`.docx`) с графиками
*   **Глобальные настройки:** Цветовая схема, параметры выбора методалиберо изменяемы из левой боковой панели

## 🛠 Технологический стек

*   **Язык:** Rust (Edition 2024).
*   **GUI Фреймворк:** [eframe/egui](https://github.com/emilk/egui) — для создания быстрого и легкого интерфейса.
*   **Графики (Real-time):** `egui_plot`.
*   **Графики (Export):** `plotters` — для рендеринга PNG-изображений высокой четкости.
*   **Работа с документами:** `docx-rs`.
