Требуется найти в соответствии с методами: Дихотомического поиска, Золотого сечения и Фибоначчи:

$$
    \max F1(x) = 3X – X3 , X  ∈ [-3; 3]
$$

$$
    \min   F2(x) = (9 - X2 )/(X2 + 2X + 3), X  ∈ [-10; -3]
$$

Предусмотреть ввод константы различимости ε и  l - допустимую конечную длину интервала неопределенности.
Провести:

1. Нахождение экстремума:
- функции F1(x)  при различных  начальных интервалах  неопределенности ( [-2; 0.5], [-1; 5], [0; 10] );
- функции F2(x) при различных  начальных интервалах неопределенности  (max на [-3;3] и [-10;10]);

2. Определение экстремума заданных функций при различных значениях константы различимости ( ε=0.1,  ε= 0.01,  ε=0.001) и допустимого конечного интервала неопределенности  ( l=0.1, l=0.01);

3. Для всех трех методов на каждой итерации предусмотреть вывод:                 номера текущей итерации; значения границ интервала                   неопределенности; значения функций  в границах интервала  неопределенности, и т.д.
                Пример  результатов вычислений:

| K   | $a_k$ | $b_k$ | $\lambda_k$ | $\mu_k$ | $F(\lambda_k)$ | $F(\mu_k)$ |

| 1   | -3.0  | 5.0   | 0.056       | 1.944   | 0.115          | 7.667      |
| 2   |       |       |             |         |                |            |
| ... |       |       |             |         |                |            |
| N   |       |       |             |         |                |            |

4. В конце вычислений выводить следующую информацию:
- оптимальное значение аргумента;
- оптимальное значение функции;
- количество вычислений функции. 

5. Построить график функции и показать на нем:
- начальные и конечные интервалы неопределенности;
- оптимальные значения аргумента и функции на каждом интервале неопределенности.
